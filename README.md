🎉 Community Event Discovery & Participation System

📌 Project Overview

This project is a data-driven web application built using Streamlit that helps users discover, analyze, and participate in community events. It provides interactive visualizations and insights about event participation trends.

---

🚀 Features

- 🔍 Filter events by Category and Location
- 📊 View Total Events and Participants
- 📈 Interactive Bar Chart (Category Analysis)
- 🥧 Pie Chart (Age Group Distribution)
- 📅 Line Chart (Monthly Trends)
- 📄 Data preview table

---

🧩 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Excel (Dataset)

⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/your-username/community-dashboard.git
cd community-dashboard

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py
